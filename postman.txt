Checklist pra enviar foto pelo Postman ✅

Gerar token no tinker:

php artisan tinker
$user = App\Models\User::find(1);
$token = $user->createToken('PostmanToken')->plainTextToken;


Abrir Postman → selecionar método POST → URL:

http://127.0.0.1:8000/api/perfil/foto


Headers → adicionar (bulk-edit):

Authorization: Bearer SEU_TOKEN_AQUI
Accept: application/json


Body → marcar form-data → adicionar:

Key: foto

Type: File

Value: escolher a imagem

Clicar em Send
